# Internet Adoption and Economic Growth Dashboard (2000–2024)

## Project Overview

This project was completed as the Week 8 Capstone Project for the AnalystLab Africa Data Analytics Internship.

The objective was to investigate the relationship between internet adoption and economic performance using the World Bank World Development Indicators (WDI) dataset.

## Project Objectives

- Analyze global internet adoption trends.
- Examine GDP per capita across countries.
- Compare regions and income groups.
- Investigate the relationship between internet adoption and GDP per capita.
- Develop an interactive Power BI dashboard.

## Dataset

Source:
World Bank – World Development Indicators (WDI)

https://datatopics.worldbank.org/world-development-indicators/

Coverage:
- 217 Countries
- 7 Regions
- 2000–2024

Indicators Used

- Individuals using the Internet (% of population)
- GDP per capita (current US$)
- Population, total

## Tools

- Python
- Pandas
- Jupyter Notebook
- Power BI

## Data Preparation

The following preprocessing steps were performed:

- Filtered relevant indicators
- Filtered years (2000–2024)
- Unpivoted yearly columns
- Merged country metadata
- Added Region and Income Group
- Removed aggregate regions
- Validated data types
- Exported cleaned dataset

## Dashboard Features

- KPI Cards
- Internet Adoption Trend
- GDP per Capita Trend
- Internet Adoption vs GDP per Capita Scatter Plot
- Top Countries by Internet Adoption
- Regional Comparison
- Income Group Comparison
- Interactive Filters

## Key Findings

- Internet adoption increased steadily between 2000 and 2024.
- High-income countries recorded the highest internet adoption.
- Countries with higher internet adoption generally exhibited higher GDP per capita.
- Significant regional disparities remain.

## Recommendations

- Expand broadband infrastructure.
- Improve internet affordability.
- Promote digital literacy.
- Support evidence-based digital policy.
